Introduction to Terrarithmetic

Terrarithmetic is a very enjoyable and exciting game, with many different things to explore. This game is intended for children between the ages 3 and 5. This game was developped by Pixelinx Interactive, and is the first of many user friendly and fun games for all ages. This is a game where the user selects a character to play as, and throughout the game this character will encounter many obstacles like opponents to battle in card games, and some hidden things that will help the character on his/her journey.
